2. Shopping Center

Create a program to manage a shop cart using OOP.

Create a Product class, that will represent a product in shopping center. 
It will have such fields as price productName quantity

Create a Shop class that will represent a place where Products are stored. 
For that, you should create an empty productList array in the constructor 
and add shopName as a parameter for the class. And add tbe method 
addNewProduct(product) to add a Product class object to the array.

As a result create Shop object with shopName and empty array productList 
Add a few products to the shop with addNewProduct(product) method. Print 
all products in the shop to the console.
